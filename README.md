## Project Tic-Tac-Toe

This is the source code for [Project Tic Tac Toe]

### Requirements
[Python 3.8](https://docs.python.org/3/whatsnew/3.8.html) or later.

### Run the game

+ _install third-party packages and invoke as a module_

```bash
$ pip install -r requirements.txt
$ python -m tictactoe
```
